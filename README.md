# arrayCrossFromX
Цель задания.

Закрепить навыки работы с массивами, используя двумерный массив.

Что нужно сделать.

Создайте с помощью циклов двумерный массив строк.

При его распечатке в консоли должен выводиться крестик из X:

	_x___x_
	
	__x_x__

	___x___

	__x_x__

	_x___x_

	x_____x
